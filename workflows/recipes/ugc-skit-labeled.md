# Recipe: UGC-Style Skit (LABELED FICTION)

**Goal:** a speaking-character skit in UGC energy — a fictional founder
mid-overbuild-spiral — for engagement, WITHOUT crossing into fake social proof.
**When to use:** "skit", "UGC-style video", humor/relatability slots.

## ⚠️ The hard rule (why this recipe is last-priority)
The 2026 meta-trend is ambiguous-authenticity AI UGC. **We deliberately break
the ambiguity:**
- Caption labels it: "dramatization" / clearly-fictional framing
- Visual grade stays on-brand (NOT raw-phone-camera cosplay)
- NEVER presented as a real user, review, or testimonial
- If a piece could be mistaken for a real customer, it does not ship — no exceptions
This is theater, and it must read as theater. (Kit rule: social proof is
user-supplied verbatim or it doesn't exist.)

## The plan
1. Script one 8s beat (`/write-copy`): a single relatable moment played big
   ("I just need ONE more feature before launch" — 3am, 14th coffee).
   `reviewScript` for restrictions.
2. Character: reuse a locked fictional character (e.g. `char-builder`) via
   `generateSpeakingVideoFromImage`, or `generateSpeakingVideo` for a new one
   (register if it works). Veo native lip-sync + voice. 8s ≈ $0.80.
   (When OPENROUTER_API_KEY is configured, Seedance offers stronger lip-sync.)
3. Caption with the fiction label + package reels/tiktok.

## Budget
$0.80–1.60/piece. Occasional format — run only when a skit idea is genuinely
funny; forced skits read as ads with extra steps.

## Success signals
Comments quoting the line back. If viewers ask "is this real?", the label
wasn't loud enough — fix the caption before the next one.
