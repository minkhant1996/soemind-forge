# Social Media Video Content

> Scroll-stopping content optimized for TikTok, Reels, Shorts, and feeds.

---

## When to Use

- TikTok videos
- Instagram Reels
- YouTube Shorts
- LinkedIn videos
- Twitter/X videos
- Facebook Reels

**Ideal Duration**: 15-60 seconds

---

## Platform Specifications

| Platform | Aspect Ratio | Duration | Hook Window |
|----------|--------------|----------|-------------|
| TikTok | 9:16 | 15-60s | 0.5s |
| Instagram Reels | 9:16 | 15-90s | 1s |
| YouTube Shorts | 9:16 | 15-60s | 1s |
| Instagram Feed | 1:1 or 4:5 | 15-60s | 1s |
| LinkedIn | 1:1 or 16:9 | 30-120s | 3s |
| Twitter/X | 16:9 or 1:1 | 15-140s | 2s |

## Content Formats

### The "POV" Format

**Structure**: First-person perspective storytelling

```
POV: You're a [ROLE] who just discovered [THING]

CLIP 1 (5s): The setup/discovery
CLIP 2 (10s): The reaction/exploration
CLIP 3 (5s): The payoff/result
```

**Example Prompt**:
```
POV shot from a person's perspective looking at their laptop screen.
Hands visible typing on keyboard.
Screen shows [DISCOVERY MOMENT].
Natural room lighting, casual home office setting.
iPhone camera aesthetic, authentic and relatable.
Vertical 9:16 aspect ratio.
```

---

### The "Day in the Life" Format

**Structure**: Follow someone through a process

```
MORNING (5s): Starting the task
MID-DAY (10s): Working through it
EVENING (5s): Completion/results
```

**Style**: Time-lapse feel, aesthetic montage

---

### The "Before/After" Format

**Structure**: Dramatic transformation

```
BEFORE (8s): The problem state
TRANSITION (2s): Quick cut or morph
AFTER (8s): The solution state
```

**Example Prompts**:

Before:
```
Messy desk with scattered papers, old monitor, tangled cables.
Harsh overhead fluorescent lighting.
Cluttered, chaotic, overwhelming energy.
Camera slowly pans across the mess.
Authentic, slightly shaky handheld feel.
Vertical 9:16 format.
```

After:
```
Same desk now minimal and organized, ultrawide monitor, cable-managed.
Soft golden hour light from window.
Plant, coffee, clean aesthetic.
Same camera movement as before for contrast.
Satisfying, aspirational, Instagram-worthy.
Vertical 9:16 format.
```

---

### The "Tutorial/Tip" Format

**Structure**: Quick, valuable how-to

```
HOOK (2s): "Here's how to [THING]"
STEP 1 (5s): First action
STEP 2 (5s): Second action
STEP 3 (5s): Third action
RESULT (3s): Final outcome
```

---

### The "Trend Reaction" Format

**Structure**: React to popular content/sound

```
SETUP (3s): Show the trend/prompt
REACTION (10s): Your unique take
PAYOFF (5s): Unexpected result
```

## Hook Techniques (First 1-3 seconds)

### Visual Hooks

| Hook Type | Description | Prompt Keywords |
|-----------|-------------|-----------------|
| **Pattern Interrupt** | Unexpected visual | "suddenly", "unexpected", "surprising" |
| **Direct Address** | Person looks at camera | "looking directly at camera", "eye contact" |
| **Action in Progress** | Start mid-action | "mid-motion", "already happening" |
| **Text on Screen** | Bold statement | Include text in prompt |
| **Satisfying Visual** | ASMR-like moment | "satisfying", "smooth", "perfect" |
| **Conflict/Tension** | Something going wrong | "struggling", "failing", "breaking" |

### Hook Prompt Examples

**Pattern Interrupt**:
```
Extreme close-up of coffee being poured in slow motion.
Suddenly interrupted by a laptop notification sound visual.
Dramatic lighting shift from warm to cool blue.
Vertical format, cinematic but casual.
```

**Direct Address**:
```
Young professional looking directly at camera with knowing smile.
Modern office background blurred.
Natural lighting, authentic feel.
About to share a secret, engaging expression.
Vertical 9:16, iPhone quality.
```

**Satisfying Visual**:
```
Perfectly aligned code on screen, then auto-formatting triggers.
Lines of code smoothly rearranging into perfect order.
Satisfying visual ASMR moment.
Clean syntax highlighting, dark mode editor.
Vertical format, screen recording aesthetic.
```

## Platform-Specific Styles

### TikTok Aesthetic
```
- Raw, authentic, unpolished
- iPhone camera quality
- Natural/available lighting
- Text overlays expected
- Fast cuts, trending sounds
- POV shots popular
```

**Prompt modifier**: `iPhone quality, authentic, slightly shaky handheld, natural lighting, casual atmosphere`

### Instagram Reels Aesthetic
```
- More polished than TikTok
- Good lighting important
- Aesthetic color grading
- Clean transitions
- Aspirational lifestyle
```

**Prompt modifier**: `Instagram aesthetic, well-lit, color graded, aspirational, lifestyle quality`

### YouTube Shorts Aesthetic
```
- Educational content works well
- More produced feel acceptable
- Tutorial style popular
- Text/graphics common
```

**Prompt modifier**: `YouTube quality, educational, clear visuals, well-composed, informative`

### LinkedIn Style
```
- Professional context
- Business casual appearance
- Office environments
- Data/results focused
- Thought leadership tone
```

**Prompt modifier**: `professional, corporate quality, business environment, LinkedIn appropriate, polished`

## Trending Content Formulas

### The "5 Things" List
```
INTRO (3s): "5 [THINGS] that will [BENEFIT]"
ITEM 1 (5s): Quick visual + text
ITEM 2 (5s): Quick visual + text
ITEM 3 (5s): Quick visual + text
ITEM 4 (5s): Quick visual + text
ITEM 5 (5s): Quick visual + text
CTA (2s): "Follow for more"
```

### The "Stop Doing This"
```
HOOK (3s): "Stop [BAD PRACTICE]"
WRONG WAY (8s): Show the mistake
RIGHT WAY (8s): Show the solution
RESULT (5s): Show the difference
```

### The "What I Wish I Knew"
```
HOOK (3s): "What I wish I knew about [TOPIC]"
LESSON 1 (8s): First insight
LESSON 2 (8s): Second insight
LESSON 3 (8s): Third insight
CTA (3s): "Save this for later"
```

## Text Overlay Best Practices

### Placement (9:16 vertical)
```
┌─────────────────┐
│    Safe zone    │
│  for top text   │
│                 │
│                 │
│  MAIN CONTENT   │
│     AREA        │
│                 │
│                 │
│   Safe zone     │
│ for bottom text │
│  (above UI)     │
└─────────────────┘
```

### Text Prompt Tips
When you need text in the video, describe it:
```
Bold white text appears: "THIS CHANGES EVERYTHING"
Text positioned in upper third of frame.
Drop shadow for readability.
Sans-serif, modern font style.
```

## Example: Tech Tip Reel

**Topic**: AI Coding Assistant
**Duration**: 25 seconds
**Platform**: Instagram Reels

**Clip 1 (5s) - Hook**:
```
Close-up of hands dramatically dropping onto keyboard.
Screen shows intimidating wall of code.
Text overlay: "Me before AI coding tools"
Stressed energy, harsh blue light.
Vertical 9:16, iPhone aesthetic.
```

**Clip 2 (8s) - Problem**:
```
POV looking at screen full of red error messages.
Hands rubbing face in frustration visible at edge of frame.
Slight camera shake showing stress.
Dark room, monitor glow only.
Relatable developer moment.
```

**Clip 3 (8s) - Solution**:
```
Same POV, now screen shows AI suggestions appearing.
Beautiful code completion in green/cyan.
Hands frozen in amazement.
Lighting noticeably warmer.
Text overlay: "After discovering [TOOL]"
```

**Clip 4 (4s) - CTA**:
```
Developer smiling at camera, thumbs up.
Clean desk, good lighting.
Text overlay: "Link in bio"
Friendly, authentic energy.
```
