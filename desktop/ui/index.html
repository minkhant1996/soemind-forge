Welcome to SoeMind Forge

This wizard installs everything the AI content studio needs — dependencies, the toolkit itself, your API keys, and the agent skills — then launches your AI assistant.

Skills, your projects, and all generated content live together inside the studio folder you pick in a moment — nothing is scattered across your system.

System check

Checking the tools the studio needs. Required items must be green to continue.


    


    
    

      
Studio folder

      
Where should the studio live? If the folder is empty, the toolkit is downloaded into it;
      if it already contains SoeMind Forge, it's used as-is.

      

        
        
      

      

      
      

    


    
    

      
API keys

      
Keys are stored only on this machine, in the studio's .env file.

      

        
        
      

      

        
        
      

      

        
        
      

      

    


    
    

      
Install & build

      
Installing npm dependencies, building the Gemini and workflow modules, and installing
      the agent skills. This runs the studio's own setup script and can take a few minutes on first run.

      
      

      

    


    
    

      
Health check

      
Running the studio's doctor — it verifies API keys, builds, and optional tools.

      
      

      

    


    
    

      
Ready to create

      
Everything is installed. Launch opens your AI assistant in a terminal inside the
      studio folder — just tell it what to make.
