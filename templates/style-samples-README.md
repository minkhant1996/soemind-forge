# Style Samples Guide

> This file is copied to `projects/{name}/style-samples/README.md`
> Add your reference content here for agent to match your style.

---

## How Style Samples Work

1. **You add samples** → Agent analyzes them
2. **Agent learns your style** → Applies to generated content
3. **No samples?** → Agent uses best practices for engaging content

---

## Folder Structure

```
style-samples/
├── README.md                 # This file
│
├── video-examples/           # Videos you like
│   ├── example-1.mp4
│   ├── example-2.mp4
│   └── notes.md              # What you like about them
│
├── image-examples/           # Image styles you like
│   ├── example-1.png
│   ├── example-2.jpg
│   └── notes.md
│
├── caption-examples/         # Text/copy style
│   └── captions.md           # Paste example captions here
│
└── audio-examples/           # Voice/music style
    ├── voice-example.mp3
    └── notes.md
```

## What to Add

### Video Examples
- Competitor ads you like
- Viral videos in your niche
- Your past successful videos
- Style references (even different industry)

**In notes.md, describe:**
```
example-1.mp4:
- Like the hook style
- Color grading is perfect
- Pacing too fast, slow down for us

example-2.mp4:
- Love the transitions
- Dialogue style matches our brand
```

### Image Examples
- Graphics you like
- Carousel designs
- Color palettes
- Typography styles

**In notes.md, describe:**
```
example-1.png:
- Clean minimal style
- Font choice is perfect
- Use this color scheme
```

### Caption Examples
Paste captions you love:

```markdown
# Caption Style Examples

## Example 1 (Engaging hook)
"Stop scrolling. This changed everything for me..."

## Example 2 (Educational)
"3 things I wish I knew before..."

## Example 3 (Our past post that worked)
[Paste your successful caption]
```

### Audio Examples
- Voice tone you want
- Background music style
- Podcast clips you like

## Notes File Template

Create `notes.md` in each folder:

```markdown
# [Folder] Style Notes

## What We Like
- [Element 1]
- [Element 2]

## What We Don't Want
- [Element to avoid]
- [Element to avoid]

## Specific References
filename.ext:
- [What to copy]
- [What to change]
```

## If No Samples

**Agent will:**
1. Use platform best practices
2. Create engaging, attention-catching content
3. Follow your project.md and brand.md guidelines
4. Optimize for the target audience

**Still recommended to add:**
- At least 1-2 video examples
- Your brand colors/fonts
- 3-5 caption examples

---

## Tips

1. **Quality over quantity** - 3 good examples > 10 random ones
2. **Explain why** - Notes help agent understand your taste
3. **Include failures** - "Don't do this" is also helpful
4. **Update over time** - Add new references as you find them
