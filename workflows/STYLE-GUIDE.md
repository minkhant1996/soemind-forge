# Visual Style Guide

> **MANDATORY:** Reference this guide when choosing visual styles for images and videos.
> The agent MUST suggest appropriate styles based on content type, audience, and brand.

---

## How to Use This Guide

1. **Identify the content goal** (educate, entertain, sell, inspire)
2. **Know the target audience** (age, culture, platform)
3. **Match to brand personality** (playful, professional, edgy, calm)
4. **Select 1-2 complementary styles** (don't mix conflicting aesthetics)
5. **Apply style keywords to prompts** consistently across all assets

## Platform-Specific Recommendations

### TikTok / Reels / Shorts
- **Best styles:** Lo-fi, Meme, Glitch, Y2K, Aesthetic, Bold 2D
- **Avoid:** Overly polished corporate, slow-paced, text-heavy

### Instagram Feed
- **Best styles:** Minimal, Editorial, Lifestyle, Aesthetic, Flat Illustration
- **Avoid:** Low quality, cluttered, off-brand

### LinkedIn
- **Best styles:** Professional, Minimal, Isometric, Clean 3D, Infographic
- **Avoid:** Meme styles, overly casual, glitch effects

### YouTube Thumbnails
- **Best styles:** Bold Pop Art, High Contrast, Expressive Faces, Cinematic
- **Avoid:** Small text, subtle colors, minimalist

### Twitter/X
- **Best styles:** Meme, Bold Graphics, Infographic, Quick Motion
- **Avoid:** Long-form, subtle, text-heavy images
