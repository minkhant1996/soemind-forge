SoeMind Forge
AI Content Studio — coding agent များအတွက်

ရိုးရှင်းသော အင်္ဂလိပ်စာဖြင့် AI ဗီဒီယို (Veo 3.1)၊ ပုံ (Nano Banana)၊ အသံသွင်းချက်၊ ဂီတနှင့် ရက် ၃၀ content ပြက္ခဒိန်များကို Claude Code, Codex CLI သို့မဟုတ် Gemini CLI အတွင်း၌ပင် ဖန်တီးပါ။ Budget သတိရှိ၊ brand ကိုက်ညီ၊ တင်ရန်အသင့်။

ဖန်တီးနိုင်တာများ ကြည့်ရန် GitHub တွင် ကြည့်ရန် ၁၀ မိနစ်ဖြင့် စတင်ရန်

ရုပ်ရှင်ဆန်သော ဖလင်များ · Kinetic စာသား Reels · ဗီဒီယိုများ · Carousel များ · အသံသွင်း · ဂီတ · ရက် ၃၀ ပြက္ခဒိန်များ

ရိုးရိုး prompt ရေးတာထက် ဘာကြောင့် သာလွန်သလဲ

AI content workflow တစ်ခုအတွက် တကယ်လိုအပ်သော ကာကွယ်ချက်များ။

  • ငွေကုန်တာ သဘောတူမှသာ ဖြစ်သည်။ project တိုင်းတွင် budget အကန့်အသတ်နှင့် အသုံးစရိတ်မှတ်တမ်း ရှိသည်။ agent သည် မဖန်တီးမီ ခန့်မှန်းကုန်ကျစရိတ်ကို ပြပြီး ဈေးကြီးသည်များ မ render ခင် ဈေးသက်သာစွာ preview ပြသည်။
  • ကိုက်ညီမှုကို မျှော်လင့်တာမဟုတ်၊ တည်ဆောက်ထားတာ။ ဇာတ်ကောင်များ၊ ကုန်ပစ္စည်းများ၊ logo များနှင့် အသံများကို တစ်ကြိမ်ဖန်တီးပြီး asset registry တွင် သော့ခတ်ကာ ID ဖြင့် ပြန်သုံးသည် — campaign တစ်ခုလုံးတွင် တူညီသော မျက်နှာ။
  • စာသား ဘယ်တော့မှ မပျက်။ Slide များနှင့် reel များတွင် ပီပြင်သော ဖောင့်အစစ်များဖြင့် pixel-perfect စာလုံးအလှ (Remotion) ကို local တွင် $0 ဖြင့် render လုပ်သည်။
  • တင်ရန်အသင့် ထုတ်ပေးသည်။ platform အလိုက် အရွယ်အစားမှန်ကန်သော media၊ စာလုံးကန့်သတ်ချက်အတွင်း caption များ၊ hashtag များနှင့် alt text — render ဖိုင်အထွေထွေဖိုင်တွဲ တစ်ခုမဟုတ်ပါ။
  • အထောက်အထားက အစစ်ပဲ။ agent သည် testimonial၊ စာရင်းအင်း သို့မဟုတ် press များ လုပ်ကြံဖန်တီးခြင်းကို လုံးဝတားမြစ်ထားသည်။ သင့်ကိန်းဂဏန်းများ အတိအကျ၊ မဟုတ်လျှင် ဘာမှမလုပ်။

Kinetic စာသား Reels

9:16 စာလုံးအလှ reel များ — ဆင့်ကဲ animate လုပ်ထားသော စာကြောင်းများနှင့် $0 စာသား rendering။

သင့်အသံသို့ Lip-Sync

ဇာတ်ကောင်ပုံ + သင့်အသံသွင်းချက် → ပါးစပ်က အသံကို လိုက်လှုပ်သော စကားပြောဗီဒီယို။

ကုန်ပစ္စည်းဓာတ်ပုံများ

ကုန်ပစ္စည်းဓာတ်ပုံအစစ်တစ်ပုံမှ e-commerce preset ၂၆ ခု — packshot၊ lifestyle၊ ရာသီအလိုက်။

ပုံများနှင့် Carousel များ

Thumbnail များနှင့် slide များစွာပါ carousel များ — pixel-perfect စာလုံးအလှဖြင့်။

အသံသွင်းနှင့် ဂီတ

မြင်ကွင်းအလိုက် narration အပြင် နောက်ခံဂီတနှင့် jingle များ။

ရက် ၃၀ Content ပြက္ခဒိန်များ

အစီအစဉ်အပြည့်အစုံနှင့် campaign recipe များ — တစ်ရက်ပြီးတစ်ရက် ထုတ်လုပ်။

ဘယ်လို အလုပ်လုပ်သလဲ

ရိုးရှင်းသော အင်္ဂလိပ်စာဖြင့် ဖော်ပြပါ။ agent က စီစဉ်၊ ဈေးဖြတ်၊ ဖန်တီးပြီး ပေါင်းစပ်ပေးသည် — သင်က အတည်ပြုရုံပါ။

You ကျွန်တော့် ကော်ဖီ brand အတွက် ၃၀ စက္ကန့် ရုပ်ရှင်ဆန်တဲ့ ဇာတ်လမ်းဖလင် လုပ်ပေးပါ ☕
Claude ရပါပြီ — pre-flight အရင်လုပ်နေပါတယ်။ brand နာမည်၊ ဘယ်သူ့အတွက်လဲ၊ ပြီးတော့ သင့် budget အကန့်အသတ်က ဘယ်လောက်လဲ။
You Brooke's Coffee ပါ၊ အလုပ်များတဲ့ founder တွေအတွက်။ $5 မှာ ကန့်သတ်ပါ။
Claude ရပါပြီ။ ဒါက အစီအစဉ်ပါ — ခန့်မှန်း $3.65၊ သင့်အကန့်အသတ်အောက်။ အခုတည်ဆောက်နေပါပြီ။ ↳ createProject brookes-coffee/ + brand.md ↳ generateImage founder keyframe → char-founder အဖြစ် သော့ခတ် ↳ registerAsset char-founder → assets.yaml (မြင်ကွင်းတိုင်း ပြန်သုံး) ↳ writeScript မြင်ကွင်း ၄ ခု ဇာတ်လမ်း + လိုက်လျောညီထွေ narration ↳ generateVideoFromImage ×4 Veo 3.1 clip များ၊ တူညီသောမျက်နှာ ↳ generateVoiceover မြင်ကွင်းအလိုက် ပြောဆိုမှု ↳ assembleFinal clip များ + VO + ဂီတ ပေါင်းစပ် → final.mp4
Claude ပြီးပါပြီ။ ၃၂ စက္ကန့် ဖလင် ပေါင်းစပ်ပြီး၊ ကုန်ကျ $3.62၊ သင့်မှတ်တမ်းတွင် သွင်းထားပြီးပါပြီ။ 9:16 + 16:9 ထုပ်ပိုးပြီး၊ caption နှင့် hashtag များပါ — တင်ရန်အသင့်။ ✅
1

သင်က ဖော်ပြသည်

ရိုးရှင်းသော အင်္ဂလိပ်စာ — "ကျွန်တော့် ကော်ဖီ brand အတွက် ရုပ်ရှင်ဆန်တဲ့ ဇာတ်လမ်းဖလင်။" prompt engineer လုပ်စရာ မလို။

2

စီစဉ်ပြီး ဈေးဖြတ်

Pre-flight က brand၊ budget နှင့် asset များကို ဖြေရှင်းပြီး၊ မသုံးစွဲမီ ကုန်ကျစရိတ်ကို ဖော်ပြသည်။

3

ဇာတ်ကောင် သော့ခတ်

founder ကို တစ်ကြိမ်ဖန်တီးပြီး asset registry တွင် သိမ်းသည် — မြင်ကွင်းတိုင်း ID ဖြင့် ပြန်သုံးသည်။

4

မြင်ကွင်းများ ဖန်တီး

Veo 3.1 က clip တစ်ခုစီကို သော့ခတ်ထားသော keyframe မှ render လုပ်သဖြင့် shot တိုင်းတွင် တူညီသောမျက်နှာ တည်တံ့သည်။

5

အသံနှင့် ဂီတ

မြင်ကွင်းအလိုက် narration နှင့် ဂီတ layer ကို ဖန်တီးပြီး cut နှင့် အချိန်ကိုက်ညှိသည်။

6

ပေါင်းစပ်

Clip များ၊ အသံသွင်းနှင့် ဂီတကို transition များဖြင့် ဖလင်တစ်ခုအဖြစ် ချုပ်စပ် — platform အလိုက် တင်ရန်အသင့်။

အမြန်စတင်ရန်
