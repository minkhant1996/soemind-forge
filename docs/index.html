Why it beats raw prompting

The guardrails an AI content workflow actually needs.

  • Money moves only with consent. Every project has a budget cap and spend ledger; the agent shows estimated cost before generating and previews cheap before rendering expensive.
  • Consistency is engineered, not hoped for. Characters, products, logos, and voices are generated once, locked in an asset registry, and reused by ID — the same face across a whole campaign.
  • Text never garbles. Slides and reels get pixel-perfect typography rendered locally with real fonts (Remotion), for $0.
  • It ships upload-ready. Correctly sized media per platform, captions within char limits, hashtags, and alt text — not a folder of loose renders.
  • Proof stays real. The agent is hard-blocked from inventing testimonials, stats, or press. Your numbers, verbatim, or nothing.

Quick start

From zero to your first generated image in about 10 minutes.

git clone https://github.com/minkhant1996/agent-content-studio
cd agent-content-studio
./setup.sh          # macOS / Linux  (setup.bat on Windows)

# add your free Google AI Studio key to .env, then:
claude              # loads project context automatically
> Create a TikTok ad for my coffee brand
